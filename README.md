# stroke_predictor_ui
ML model for stroke prediction with UI

This is a simple project of Machine Learning, with a publicly availabe dataset of clinical data
used in various places of the internet.

The dataset used is relative to stroke occurence in a general population, and contains sociodemographic data
as well as some clinical data, it is anonymized and it was subjected to some minor pre treatment because it's a relative
dirty dataset.

The base model for this machine learning project is the Random Forest Classifier.


IT's a work in progress and it's only a hobby and experimentation tool so it's bound to be put down eventually or migrated to
another service.

It currently runs on render as a webservice with a UI coded by AI with human supervision.

The rest of the code and process were made by human hands with some guidance of online tutorials and guides.



Don't take the results of the predictions as HEALTH ADVICE.
